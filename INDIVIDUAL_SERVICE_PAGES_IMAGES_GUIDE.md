## üöÄ How to Add Images

### Step 1: Prepare Images

1. Resize to recommended sizes
2. Optimize for web (use Squoosh.app)
3. Save to appropriate service directory

### Step 2: Update Service Page

1. Uncomment the imports in the service page file
2. Uncomment Image components
3. Remove placeholder divs

### Step 3: Test

1. Build: `npm run build`
2. Check images display correctly
3. Verify responsive behavior
